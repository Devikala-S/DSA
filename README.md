# Devikala S — UI/UX Portfolio

Premium, recruiter-focused portfolio website (typography-driven, minimal luxury aesthetic).

## Run
```bash
npm install
npm run dev
```

Open the URL shown in the terminal (usually `http://localhost:5173`).

## Build
```bash
npm run build
npm run preview
```

## Edit placeholders
- Replace `devikala@example.com` in `index.html` with your real email.
- Update LinkedIn/GitHub URLs in `index.html`.
- Add a real resume download link (currently `#`).
